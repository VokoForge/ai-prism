## 平台发布指南

### 小红书 (Xiaohongshu/RED)

| 项目 | 规格 |
|------|------|
| 图片尺寸 | 1080×1440px (3:4) |
| 格式 | SVG → 导出 PNG/JPG |
| 数量 | 3-9张/篇 |
| 标题 | 20字以内，数字+情绪词 |
| 正文 | 300-500字，emoji点缀 |
| 标签 | 5-8个，#AI #开源 #创业 等 |
| 发布时间 | 12:00-13:00 / 20:00-22:00 |

**内容适配规则：**
- 视觉优先：每张卡片一个核心观点
- 数字锚点：用具体数据增强可信度
- 口语化：避免学术腔，像朋友聊天
- 封面卡必须最抓眼球

## 内容适配原则

1. **一源多端**：同一内容源，按平台特性适配，不是简单复制
2. **视觉先行**：小红书重卡片，公众号重排版，知乎重深度，Twitter重精炼
3. **数据锚点**：每个平台版本都必须保留关键数据（stars数、性能倍数、成本数字）
4. **CTA一致**：所有平台结尾统一引导至 AI棱镜 仓库
5. **语言适配**：国内平台中文，LinkedIn英文，Twitter中英皆可
